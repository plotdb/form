# Form Design

這裡定義表單元件設計的基本結構.

## Form Data

 - title: '...'
 - desc: '...'
 - criteria: []
 - config: {}


## Form Structure

 - form-lib
   - blocks
   - types
   - rulesets
   - block
     - type ( name, or inline )
   - type
     - name
     - attrs
       - <attr-name>:
         - ruleset ( name, or inline )
         - get 
   - ruleset
     - name
     - sanity-check
     - convert
     - default-operation
     - operations
       - <operation-name>:
         - args
         - func
   - criteria

 - form-def
   - block
   - data
